NBP Currency Rates for Odoo

Module extends built-in live currency module to use NBP (National Polish Bank) REST API to download current exchange rates.

It downloads data from table A (middle exchange rates for popular currencies) then (if needed) from table B (middle exchange rates for other currencies).

Module fetches rate table that is active at the moment of download.
